# Together 开发进度

- 持续打磨单人 Todo 的 Today 主链路，优先提升创建、查看、完成、推迟、详情编辑与首页交互手感。

## 最近完成
- 首页无任务空态已改成更轻的内联表达：去掉独立卡片与状态标签，保留备注式文案、居中的新建任务按钮，以及已完成入口。
- Home 侧已支持从空态直接进入新建任务入口，不需要再依赖底部 Dock 才能创建。
